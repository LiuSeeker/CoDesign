Eletrônica

Mini-estação meteorológica

Informaçāoes básicas:

  • Curso: Instrumentação e Medição
  • Como? Através de conhecimentos básicos de eletrônica para o ajuste dos sensores
  • Quando? 1˚ semestre de 2017

Descrição:

Utilização de sensores de umidade , pressão , temperatura e altura para a construção de uma mini estação meteorológica, em que precisavamos calibrar os diferentes sensores para obtermos dados coerentes e a sua visualização dos dados no display-lcd.

Os sensores enviavam ao Arduino em periodos pré-determinados no código os dados calculados que representavam diferentes d.d.p(diferencia de potencial) através de variações no sistema , como quando a temperatura varia. Por esse motivo utilizamos um codigo no Arduino para convertermos os dados encontrados nos sensores para as escalas que estamos acostumados,como metros para a altura.
